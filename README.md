# 🎰 Sayisal Loto Oyunu

Bu proje, C# programlama dili kullanılarak geliştirilmiş basit bir sayısal loto oyunudur.

## 🚀 Proje Amacı
Kullanıcının girdiği sayılar ile sistem tarafından rastgele üretilen sayıları karşılaştırarak sonucu belirleyen bir uygulamadır.

## ⚙️ Özellikler
- Rastgele sayı üretimi
- Kullanıcıdan veri alma
- Girilen sayılar ile sistem sayılarının karşılaştırılması
- Sonuçların ekrana yazdırılması

## 🛠️ Kullanılan Teknolojiler
- C#
- .NET

## 📌 Nasıl Çalışır?
1. Kullanıcı belirli sayılar girer
2. Sistem rastgele sayılar üretir
3. Girilen ve üretilen sayılar karşılaştırılır
4. Eşleşme durumuna göre sonuç gösterilir
